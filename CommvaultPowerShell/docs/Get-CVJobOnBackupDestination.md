# Get-CVJobOnBackupDestination

## SYNOPSIS
Get the list of Jobs for selected backupDestinationId.

## SYNTAX

### Get (Default)
```
Get-CVJobOnBackupDestination -BackupDestinationId <Int64> [-AgedData <Int64>] [-BackupLvl <Int64>]
 [-Clients <String>] [-CopyState <Int64>] [-EndTime <Int64>] [-StartTime <Int64>] [-View <String>]
 [<CommonParameters>]
```

### GetViaIdentity
```
Get-CVJobOnBackupDestination -InputObject <ICommvaultPowerShellIdentity> [-AgedData <Int64>]
 [-BackupLvl <Int64>] [-Clients <String>] [-CopyState <Int64>] [-EndTime <Int64>] [-StartTime <Int64>]
 [-View <String>] [<CommonParameters>]
```

## DESCRIPTION
Get the list of Jobs for selected backupDestinationId.
