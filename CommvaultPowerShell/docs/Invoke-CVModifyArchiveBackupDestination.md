# Invoke-CVModifyArchiveBackupDestination

## SYNOPSIS
Modify Backup Destination for a Plan

## SYNTAX

### ModifyExpanded (Default)
```
Invoke-CVModifyArchiveBackupDestination -BackupDestinationId <Int64> -PlanId <Int64> [-EnableDataAging]
 [-NewName <String>] [-OverrideRetentionSettings] [-RetentionPeriodDays <Int64>] [-SourceCopyId <Int64>]
 [-SourceCopyName <String>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Modify
```
Invoke-CVModifyArchiveBackupDestination -BackupDestinationId <Int64> -PlanId <Int64>
 -Body <IUpdateArchivePlanBackupDestination> [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### ModifyViaIdentity
```
Invoke-CVModifyArchiveBackupDestination -InputObject <ICommvaultPowerShellIdentity>
 -Body <IUpdateArchivePlanBackupDestination> [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### ModifyViaIdentityExpanded
```
Invoke-CVModifyArchiveBackupDestination -InputObject <ICommvaultPowerShellIdentity> [-EnableDataAging]
 [-NewName <String>] [-OverrideRetentionSettings] [-RetentionPeriodDays <Int64>] [-SourceCopyId <Int64>]
 [-SourceCopyName <String>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```
