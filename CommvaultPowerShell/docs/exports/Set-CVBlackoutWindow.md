# Set-CVBlackoutWindow

## SYNOPSIS
Modify the properties of an existing Blackout Window

## SYNTAX

### UpdateExpanded (Default)
```
Set-CVBlackoutWindow [-Properties] <Object> -BlackoutWindowId <Int64> [-BackupOperations <String[]>]
 [-BetweenDateEnd <Int64>] [-BetweenDateStart <Int64>] [-CompanyId <Int64>] [-CompanyName <String>]
 [-Days <String[]>] [-DoNotSubmitJob] [-Force] [-NewName <String>] [-Time <IStartEnd[]>] [-Weeks <String[]>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Set-CVBlackoutWindow [-Properties] <Object> -InputObject <ICommvaultPowerShellIdentity>
 [-BackupOperations <String[]>] [-BetweenDateEnd <Int64>] [-BetweenDateStart <Int64>] [-CompanyId <Int64>]
 [-CompanyName <String>] [-Days <String[]>] [-DoNotSubmitJob] [-Force] [-NewName <String>]
 [-Time <IStartEnd[]>] [-Weeks <String[]>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

Modify the properties of an existing Blackout Window
