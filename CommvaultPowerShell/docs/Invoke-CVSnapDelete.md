# Invoke-CVSnapDelete

## SYNOPSIS
Deleting an unmounted or created snapshot from Storage Arrays

## SYNTAX

### SnapExpanded (Default)
```
Invoke-CVSnapDelete -ArrayId <Int64> [-Volumes <IVolumeSnap[]>] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### Snap
```
Invoke-CVSnapDelete -ArrayId <Int64> -Body <IDeleteSnapReq> [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### SnapViaIdentity
```
Invoke-CVSnapDelete -InputObject <ICommvaultPowerShellIdentity> -Body <IDeleteSnapReq> [-PassThru] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### SnapViaIdentityExpanded
```
Invoke-CVSnapDelete -InputObject <ICommvaultPowerShellIdentity> [-Volumes <IVolumeSnap[]>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Deleting an unmounted or created snapshot from Storage Arrays
