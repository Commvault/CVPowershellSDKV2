# Set-CVUser

## SYNOPSIS
Used to modify an existing user

## SYNTAX

### UpdateExpanded (Default)
```
Set-CVUser -UserId <Int64> [-Email <String>] [-Enabled] [-FullName <String>] [-NewName <String>]
 [-NewPassword <String>] [-PlanId <Int64>] [-PlanName <String>] [-UserGroupOperation <String>]
 [-UserGroups <IIdName[]>] [-UserPrincipalName <String>] [-ValidationPassword <String>] [-PassThru] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Set-CVUser -InputObject <ICommvaultPowerShellIdentity> [-Email <String>] [-Enabled] [-FullName <String>]
 [-NewName <String>] [-NewPassword <String>] [-PlanId <Int64>] [-PlanName <String>]
 [-UserGroupOperation <String>] [-UserGroups <IIdName[]>] [-UserPrincipalName <String>]
 [-ValidationPassword <String>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```
