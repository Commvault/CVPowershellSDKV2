# Stop-CVJob

## SYNOPSIS
Stops/kills the job specified by job Id.

## SYNTAX

```
Stop-CVJob [-JobId] <Int32> [<CommonParameters>]
```

## DESCRIPTION
Stops/kills the job specified by job Id.
If the job cannot be killed, this command has no effect.
