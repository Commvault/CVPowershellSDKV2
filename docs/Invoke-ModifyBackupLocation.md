# Invoke-ModifyBackupLocation

## SYNOPSIS
Modify the properties of an existing mount path

## SYNTAX

### ModifyExpanded (Default)
```
Invoke-ModifyBackupLocation -BackupLocationId <Int32> -StoragePoolId <Int32> [-Access <String>]
 [-CredentialsName <String>] [-CredentialsPassword <String>] [-Enabled] [-MediaAgentId <Int32>]
 [-MediaAgentName <String>] [-Path <String>] [-SavedCredentialsId <Int32>] [-SavedCredentialsName <String>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### Modify
```
Invoke-ModifyBackupLocation -BackupLocationId <Int32> -StoragePoolId <Int32> -Body <IUpdateBackupLocation>
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### ModifyViaIdentity
```
Invoke-ModifyBackupLocation -InputObject <ICommvaultPowerShellIdentity> -Body <IUpdateBackupLocation>
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### ModifyViaIdentityExpanded
```
Invoke-ModifyBackupLocation -InputObject <ICommvaultPowerShellIdentity> [-Access <String>]
 [-CredentialsName <String>] [-CredentialsPassword <String>] [-Enabled] [-MediaAgentId <Int32>]
 [-MediaAgentName <String>] [-Path <String>] [-SavedCredentialsId <Int32>] [-SavedCredentialsName <String>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```
